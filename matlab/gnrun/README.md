# MATLAB Newey-West Simulation Example for ICDS Roar Collab Cluster

## Overview
This directory contains a MATLAB-based simulation and analysis workflow. The example performs a Monte Carlo simulation of a Gauss-Markov (GN) process, calculates Newey-West adjusted heteroskedastic-serial consistent t-statistics, and generates visualization plots.

## Files
- `combo.submit` - SLURM submission script to run the MATLAB job in batch mode.
- `mygnrun.m` - The main MATLAB function containing the simulation logic, visualization, and helper functions for Newey-West correction.
- `figure1.jpg` / `figure2.jpg` - Output plots generated by the simulation.

## How to Run

### Submit the Job
Use `sbatch` to submit the script to the cluster scheduler:
```bash
sbatch combo.submit
```

Check Job Status
To monitor the progress of your job:
```bash
squeue --me --start
```


## Script Breakdown
combo.submit

```bash
#!/bin/bash
#SBATCH --nodes=1                     # Request 1 compute node
#SBATCH --ntasks-per-node=1           # Run a single task
#SBATCH --time=10:00                  # Set a 10-minute time limit
#SBATCH --mem=10GB                    # Allocate 10 GB of memory
#SBATCH --partition=basic             
#SBATCH --account=open                
#SBATCH --output=matlab.%j.out        # Standard output log (%j = job ID)
#SBATCH --error=matlab.%j.err         # Standard error log

# Clear environment and load the specific MATLAB version
module purge
module load matlab/R2022a

# Run MATLAB in batch mode without a GUI
# Note: "mygnrun(1000)" passes 1000 iterations to the function
matlab -batch "mygnrun(1000)"

```
