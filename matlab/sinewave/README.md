# MATLAB Sine Wave Visualization Example for ICDS Roar Collab Cluster

## Overview
This directory contains a MATLAB-based example for reading numerical data from a CSV file and generating a signal plot. This workflow is commonly used for visualizing results generated by other tools or previous simulation steps on the cluster.

## Files
- `sub_nsin_mat.submit` - SLURM submission script to execute the MATLAB visualization.
- `read_nsin.m` - MATLAB script that reads `noiseysine.csv` and exports a PNG plot.
- `noiseysine.csv` - The input data file containing coordinates for the sine wave.

## How to Run

### Submit the Job
Use `sbatch` to submit the execution script to the Roar Collab scheduler:
```bash
sbatch sub_nsin_mat.submit
```

Check Job Status
To monitor the progress of your job:
```bash
squeue --me --start
```


## Script Breakdown
sub_nsin_mat.submit

```bash
#!/bin/bash

#SBATCH --job-name=nsin_mat          # Name of the job in the queue
#SBATCH --account=open                
#SBATCH --partition=basic             
#SBATCH --nodes=1                     # Request 1 compute node
#SBATCH --ntasks-per-node=2           # Request 2 tasks on the node
#SBATCH --mem-per-cpu=1GB             # Allocate 1GB of RAM per CPU core
#SBATCH --time=00:01:00               # Set a 1-minute time limit
#SBATCH --output=nsin_mat.%j.out      # Standard output log (%j = job ID)
#SBATCH --error=nsin_mat.%j.err       # Standard error log

# Load the default MATLAB module
module load matlab

# Execute the MATLAB script in batch mode
# This runs 'read_nsin.m' without opening the GUI
matlab -batch read_nsin

```
