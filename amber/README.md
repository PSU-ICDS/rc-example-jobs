# Amber Example

This job runs a molecular dynamics simulation using pmemd following the 
tutorial presented here: https://ambermd.org/tutorials/basic/tutorial14/index.php

Input Files:
 - RAMP1.prmtop: topology file generated by LEaP
 - RAMP1_eq7.rst7: coordinate file of the equilibrated system
 - md.mdin: input file specifying the run conditions
 - amber.submit: Slurm submit script that outlines the CPU batch job
 - amber-cuda.submit: Slurm submit script that outlines the GPU enabled batch job
